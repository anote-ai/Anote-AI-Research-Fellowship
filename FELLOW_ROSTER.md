## Column Guide

| Column | What to Fill In |
|--------|----------------|
| **Full Name** | Fellow's full legal name |
| **School / Institution** | University or institution they are currently enrolled at or affiliated with |
| **Degree & Year** | e.g. "BS Computer Science, Junior", "MS AI, 1st Year", "PhD NLP, 2nd Year" |
| **Research Track** | Track ID + title from [`RESEARCH_FELLOW_MATCHING.md`](./RESEARCH_FELLOW_MATCHING.md) — e.g. "T1a: AgenticEval" |
| **Assigned Product** | Product name from [`PRODUCT_FELLOW_MATCHING.md`](./PRODUCT_FELLOW_MATCHING.md) — e.g. "Panacea" |
| **GitHub Handle** | Their GitHub username — PM to confirm repo access before Jun 3 |
| **LinkedIn** | Full LinkedIn profile URL |
| **Location** | City, State / Country |
| **Remote / In-Person** | "In-Person (NYC)", "Remote", or "Hybrid" |
