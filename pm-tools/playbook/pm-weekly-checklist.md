# PM Weekly Checklist

This is the PM's recurring operating checklist. Run it every week.

---

## Monday (Before Standup Week Begins)

- [ ] Post weekly async check-in prompt in #research-standup and #product-standup (use `fellow-check-in.md` template)
- [ ] Review last week's standup notes — identify any unresolved blockers
- [ ] Check on any open at-risk fellows from last week
- [ ] Confirm this Friday's speaker is set — did they confirm receipt of the 48-hour reminder?
- [ ] Review conference deadlines this week — any reminders due?
- [ ] Update `fellow-progress-tracker.md` with status from last week

---

## Tuesday (Research Standup Day)

- [ ] Post standup agenda in Zoom chat at 4:55 PM (use `standup-agenda.md` template)
- [ ] Open Zoom at 4:58 PM, start recording
- [ ] Facilitate standup — 30 min max
- [ ] Fill in PM notes table during standup
- [ ] Stop recording, save notes
- [ ] Post standup notes in #research-standup thread within 1 hour
- [ ] DM any fellow with a blocker to schedule async follow-up
- [ ] Update `fellow-progress-tracker.md`

---

## Wednesday (Mid-Week Check)

- [ ] Follow up on any open blockers from Tuesday
- [ ] Check GitHub activity — are fellows committing / opening PRs?
  - Quick check: github.com/anote-ai — sort by recently updated
- [ ] Confirm this Friday's speaker: room booked, Zoom ready, run sheet complete
- [ ] Draft LinkedIn speaker announcement for Thursday post (use `social-media-posts.md`)
- [ ] Any at-risk fellows to flag to Natan before Thursday?

---

## Thursday (Product Standup Day + Speaker Post)

- [ ] Post LinkedIn speaker announcement for Friday talk
- [ ] Post standup agenda in Zoom chat at 4:55 PM (use `standup-agenda.md` template)
- [ ] Open Zoom at 4:58 PM, start recording
- [ ] Facilitate product standup — 30 min max
- [ ] Fill in PM notes table during standup
- [ ] Stop recording, save notes
- [ ] Post standup notes in #product-standup thread within 1 hour
- [ ] Send weekly pulse summary to Natan (by 6 PM — use template in `standup-agenda.md`)
- [ ] Post Slack reminder for Friday talk in #fellowship-talks: "Tomorrow 1 PM: [speaker]"

---

## Friday (Fellowship Talk Day)

- [ ] Arrive WeWork NYC by 12:00 PM
- [ ] Run AV setup checklist (see `fellowship-talk-runsheet.md`)
- [ ] Meet speaker at 12:30 PM (in-person) or coordinate remote join at 12:45 PM
- [ ] Tech check by 12:45 PM
- [ ] Start recording at 12:55 PM
- [ ] Welcome and introduce speaker at 1:00 PM (use intro script in run sheet)
- [ ] Moderate Q&A 1:40–1:55 PM
- [ ] Thank speaker, wrap up by 2:00 PM
- [ ] Confirm recording saved (Zoom cloud + backup)
- [ ] Post Slack thank-you to speaker and recording link (within 2 hours)

---

## Weekend / By Sunday Night

- [ ] Edit and upload talk recording to YouTube (title: "Anote AI Fellowship Talk #[N]: [Topic] | [Speaker]")
- [ ] Share recording link in #fellowship-talks Slack
- [ ] Post LinkedIn "In case you missed it" post (use `social-media-posts.md` template)
- [ ] Archive run sheet in `pm-tools/talks/`
- [ ] Send thank-you email to speaker (use `speaker-outreach-kit.md` template)
- [ ] Add recording URL to `speaker-pipeline-tracker.md`
- [ ] Update PM log in `playbook/pm-log.md` with decisions/lessons from this week

---

## Monthly Checklist (run at start of each month)

- [ ] Review and update `fellow-progress-tracker.md` — full dashboard refresh
- [ ] Review `budget-tracker.md` — reconcile against actuals, update Natan
- [ ] Check conference deadlines — any in the next 4 weeks? Reminders sent?
- [ ] Schedule steering committee touchpoint (monthly cadence)
- [ ] Review speaker pipeline — are all remaining talks confirmed?
- [ ] Post fellow spotlight on LinkedIn (rotate through cohort)
- [ ] Check in with Natan on any structural issues, staffing, or program changes
