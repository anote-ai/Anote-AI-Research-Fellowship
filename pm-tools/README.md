# PM Tools & Templates — Anote AI Research Fellowship

Everything the Fellowship PM needs to run the summer without reinventing the wheel for each meeting or email.

---

## How to Use This Folder

1. **Before June 1** — set up your trackers (fill in fellow names, assign research tracks, confirm speaker pipeline)
2. **Each week** — run `playbook/pm-weekly-checklist.md` top to bottom
3. **Each standup** — copy the relevant block from `templates/standup-agenda.md` into Zoom chat
4. **Each Friday talk** — follow `templates/fellowship-talk-runsheet.md` exactly
5. **Each speaker interaction** — use the templates in `templates/speaker-outreach-kit.md`
6. **Every Friday night** — add a log entry to `playbook/pm-log.md`
