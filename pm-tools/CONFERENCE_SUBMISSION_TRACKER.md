## arXiv Submission Checklist

> Complete this before submitting to arXiv. Each fellow should submit to arXiv before (or simultaneously with) conference submission.

### Fellow: [Name] | Paper: [Title]

**Pre-Submission**
- [ ] Paper is in LaTeX format (preferred) or PDF
- [ ] Authors and affiliations listed correctly (include "Anote AI Research Fellowship" affiliation)
- [ ] Abstract is under 1,920 characters
- [ ] All figures have alt text / captions
- [ ] References formatted correctly (BibTeX recommended)
- [ ] No identifying info that violates double-blind requirements (if submitting to blind review first)
- [ ] Proofread final draft

**arXiv Submission Steps**
- [ ] Create / log into arXiv.org account
- [ ] Select primary category (cs.LG, cs.CL, cs.AI, etc.)
- [ ] Add secondary categories if relevant
- [ ] Upload source files (LaTeX + figures) or PDF
- [ ] Enter title, authors, abstract
- [ ] Set license (recommend CC BY 4.0)
- [ ] Submit and note the arXiv ID (e.g., arXiv:2608.XXXXX)
- [ ] Share arXiv link with PM (Rajshri) and Natan
- [ ] Update [Fellow Milestone Tracker](./FELLOW_MILESTONE_TRACKER.md)

## OpenReview Submission Checklist

> Used for ICLR, NeurIPS, and other OpenReview-hosted conferences.

### Fellow: [Name] | Conference: [Conference]

- [ ] Create OpenReview account at openreview.net
- [ ] Locate the correct venue / track (double-check year and track name)
- [ ] Complete author profile (affiliation, DBLP, Semantic Scholar links)
- [ ] Prepare submission PDF in conference template format
- [ ] Enter all co-authors and confirm they have OpenReview profiles
- [ ] Upload PDF and supplementary materials
- [ ] Fill in all required fields: title, abstract, keywords, subject areas
- [ ] Review submission preview carefully
- [ ] Submit before the deadline (aim 48 hours early)
- [ ] Record submission number
- [ ] Share submission link with PM
