# Fellow Check-In Templates

---

## Weekly Async Check-In (send Monday via Slack, 9 AM)

Post this in #research-standup or DM individually if someone's been quiet:

```
👋 Weekly check-in — [Week N], [Date]

Drop a quick update before Tuesday standup:

1. 📄 Research: What did you work on last week? Where are you in the paper?
2. 💻 Product: Any PRs opened/merged? What's your current task?
3. 🚧 Blockers: Anything slowing you down? Tools, access, feedback?
4. 🎯 This week: Top 1–2 priorities for the next 7 days.

(~5 min to fill out, keeps Tuesday standup fast)
```
